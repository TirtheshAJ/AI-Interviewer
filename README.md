## What is this project about?

The **AI Interview Simulator** is a Python-based application that helps users practice job interviews in a realistic, AI-powered environment. It uses your webcam and microphone to conduct role-based mock interviews, analyze your facial expressions and voice, and provide real-time feedback on your communication, confidence, and interview skills.

**Features:**
- Role and experience selection (e.g., Full Stack Developer, Data Scientist, HR)
- Real-time webcam and microphone integration
- Face analysis (eye contact, head movement, stability)
- Voice analysis (speech-to-text, pauses, speaking speed)
- Instant feedback after each question
- Final report with strengths and weaknesses
- Simple OpenCV-based UI

## How to fork and run

1. **Fork this repository** on GitHub ([How to fork a repo](https://en.wikipedia.org/wiki/Fork_(software_development))).
2. **Clone your fork** to your local machine:
   ```bash
   git clone https://github.com/YOUR_USERNAME/ai-interviewer.git
   cd ai-interviewer
   ```
3. **(Optional but recommended) Create a virtual environment:**
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # On Windows: .venv\Scripts\activate
   ```
4. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```
5. **Run the application:**
   ```bash
   python ai_interviewer/main.py
   ```
